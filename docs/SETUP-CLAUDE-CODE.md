# 이비서 설정 매뉴얼 — 컴퓨터 일꾼(Claude Code) 켜기

> 이비서의 **일꾼(워처)** 은 마법이 아니라 **내 컴퓨터에서 켜둔 Claude Code**(AI 비서 프로그램)입니다.
> 폰 제미나이가 "작업 메모판(`eb-calendar`)"에 적어둔 일을, 이 컴퓨터 일꾼이 **몇 분마다 들여다보며 대신 처리**합니다.
> 그래서 처음에 한 번 **이 일꾼을 깔고 켜는** 준비가 필요합니다. 아래를 그대로 따라 하세요. 비개발자도 됩니다.

---

## 큰 그림 (1분)

```
[나] --말로--> [폰 제미나이] --못 하는 일은 메모--> [eb-calendar] <--몇 분마다 확인-- [내 컴퓨터의 Claude Code = 일꾼]
                                                                                        └─ 안전하게 대신 처리 → 메모에 완료 도장
```

- **폰 쪽**(제미나이) 준비는 [README의 6단계](../README.md#6단계로-끝내는-셋업)에 있습니다.
- **이 문서**는 그중 **일꾼(컴퓨터의 Claude Code)** 을 깔고 **계속 켜두는** 방법만 따로, 더 자세히 다룹니다.

> 💡 **꼭 기억할 한 가지:** 이 일꾼(Claude Code)이 **켜져 있을 때만** 메모판의 일이 처리됩니다. 컴퓨터를 끄거나 창을 닫으면 일꾼도 쉽니다(밀린 일은 다시 켜면 이어서 합니다). 24시간 돌리는 법은 맨 아래 5단계.

---

## 준비물

- 컴퓨터 1대 (Windows 또는 Mac)
- 본인 **개인 Google 계정** (폰 제미나이와 같은 계정)
- 인터넷

---

## 1단계 — 컴퓨터 일꾼(Claude Code) 설치

**Claude Code** 는 내 컴퓨터에서 도와주는 Anthropic의 AI 비서입니다. 이비서의 일꾼이 바로 여기서 돕니다.

1. 설치 안내 페이지를 엽니다 → <https://docs.anthropic.com/en/docs/claude-code>
2. 안내대로 설치합니다(터미널에서 한 줄 설치). 설치 후 터미널에서 `claude` 라고 치면 켜집니다.
3. 처음 켜면 **로그인**하라고 합니다 — 안내대로 로그인하세요(브라우저가 열립니다).

> 컴퓨터/터미널이 처음이라 막막하면, 설치 페이지의 화면을 그대로 따라 하면 됩니다. 막히는 줄을 **복사해서 Claude Code에게 "이거 어떻게 해?"** 라고 물어도 됩니다.

---

## 2단계 — 이비서가 살 폴더 만들기

이비서의 기억과 설정이 담길 **빈 폴더**를 하나 만듭니다(예: 바탕화면에 `이비서` 폴더).

- 그 폴더를 Claude Code로 엽니다(터미널에서 그 폴더로 이동한 뒤 `claude` 실행).
- 앞으로 이 폴더가 **내 두 번째 뇌**가 됩니다.

---

## 3단계 — 이비서 깔기 (스킬 설치 + 기본 셋업)

Claude Code가 켜진 상태에서, **터미널(셸)** 에 한 줄:

```bash
npx skills@latest add jhs512/eb --all
```

그다음 **Claude Code 채팅창**에 차례로 입력합니다(각 단계는 Claude Code가 손잡고 안내합니다):

```
/eb-setup     # 기억 보드(빈 표) 만들기
/eb-gcp       # (Google 계정당 1회) 시트에 쓸 열쇠 만들기
/eb-sheets    # 내 기억을 'eb' 구글 시트로 비추기 (폰에서 지식 조회의 핵심)
/eb-github    # 개인(비공개) 백업 + 자동화
/eb-pages     # (선택) 웹페이지로 보기 (Cloudflare, 무료) — README의 아기단계 참고
```

> 자세한 설명·막힘 팁은 [README 6단계](../README.md#6단계로-끝내는-셋업)에 있습니다. 이름 기본값은 **작업큐 캘린더 `eb-calendar`**, **지식 시트 `eb`** 입니다.

---

## 4단계 — 일꾼(워처) 켜기 ★ 이 문서의 핵심

Claude Code 채팅창에:

```
/eb-queue
```

이 한 줄이 다음을 해줍니다.

1. **작업 메모판** 인 보조 캘린더 **`eb-calendar`** 를 확인/안내합니다.
   - ⚠️ 보조 캘린더 **생성은 PC 구글 캘린더 웹에서만** 됩니다(폰 앱에선 새 보조 캘린더를 못 만듭니다). [Google Calendar](https://calendar.google.com) → 좌측 **"다른 캘린더 +"** → **새 캘린더** → 이름 `eb-calendar`.
2. **일꾼(워처)** 을 켭니다 — Claude Code가 **몇 분마다(기본 4분)** 메모판을 들여다보게 만듭니다.
3. 폰 제미나이에 넣을 **비서 지침(Saved info)** 도 함께 안내합니다.

이제 일꾼이 도는 동안 Claude Code 화면에는 몇 분마다 이런 **상태 한 줄**이 찍힙니다:

```
대기 0 · 실행중 0 · 완료 0 · 실패 0 · 확인필요 0 | 오늘 처리 0
```

- **대기** = 폰에서 시켜둔, 아직 처리 안 된 일
- **실행중 / 완료 / 실패** = 일꾼이 처리 중 / 끝낸 / 실패한 일
- **확인필요** = 위험해서 **당신 승인**을 기다리는 일(예: 메일 발송)

> 일꾼이 켜져 있는 동안에는 **이 Claude Code 창을 닫지 마세요.** 닫으면 일꾼이 쉽니다.

---

## 5단계 — 컴퓨터를 꺼도 24시간 돌리기 (선택, 권장)

컴퓨터를 끄거나 창을 닫으면 일꾼이 멈춥니다. **항상 돌게** 하려면 **클라우드에서 도는 일꾼**으로 옮기면 됩니다. Claude Code 채팅창에:

```
/schedule
```

라고 하고 "이비서 워처를 클라우드에서 4분마다 돌려줘" 라고 하면, 컴퓨터를 꺼도 클라우드의 일꾼이 메모판을 처리합니다.

> **일꾼은 하나만** 켜두세요. 내 컴퓨터 일꾼과 클라우드 일꾼을 **둘 다** 켜면 같은 일을 두 번 잡으려다 꼬일 수 있습니다. 클라우드로 옮겼으면 컴퓨터 쪽 일꾼은 끄세요.

---

## 다 됐어요 — 이제 폰에 말하세요

준비가 끝나면, 일꾼은 알아서 돕니다. 당신은 **폰 제미나이에 말만** 하면 됩니다.

- "지난 회의 내용 이비서에 기억해둬"
- "그 메일 답장 보내줘" → 위험한 일이라 **확인필요**로 올라오고, 당신이 직접 승인합니다.
- "작업큐 지금 어때?" → 대기·완료·확인필요를 알려 줍니다.

폰 쪽(제미나이 앱 + Saved info) 마무리는 [README 6단계](../README.md#6단계로-끝내는-셋업)를 보세요.

---

## 자주 막히는 곳

- **상태 줄이 안 찍혀요 / 일이 처리가 안 돼요** → 일꾼(Claude Code) 창이 켜져 있는지 먼저 확인. 닫혀 있으면 다시 열고 폴더에서 `claude` 실행. 오래 자리를 비우면 5단계(클라우드)로 옮기세요.
- **폰에서 "내 기억" 조회가 비어요** → 제미나이 앱에서 **Google Drive 연결**이 켜져 있는지, 시트 이름이 **`eb`** 인지 확인(3단계 `/eb-sheets`).
- **메모판(`eb-calendar`)이 안 보여요** → 보조 캘린더라 평소엔 숨겨 둡니다. 구글 캘린더 좌측 목록에서 `eb-calendar` 체크를 켜면 보입니다. 없으면 PC 웹에서 새로 만드세요(4단계).
- **위험한 걸 시켰는데 안 해요** → 정상입니다. 메일 발송·삭제 같은 **되돌릴 수 없는 일은 자동으로 하지 않고** `확인필요`로 올려 **당신 승인**을 기다립니다.
