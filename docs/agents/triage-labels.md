# 트리아지 라벨

스킬들은 다섯 가지 표준 트리아지 역할로 이야기합니다. 이 파일은 그 역할들을 이 저장소 이슈 트래커에서 실제로 쓰는 라벨 문자열로 매핑합니다.

| mattpocock/skills의 라벨 | 우리 트래커의 라벨 | 의미                          |
| ------------------------ | ------------------ | ----------------------------- |
| `needs-triage`           | `needs-triage`     | 메인테이너의 평가 필요        |
| `needs-info`             | `needs-info`       | 작성자의 추가 정보 대기       |
| `ready-for-agent`        | `ready-for-agent`  | 완전히 명세됨, AFK 에이전트 준비 완료 |
| `ready-for-human`        | `ready-for-human`  | 사람의 구현 필요              |
| `wontfix`                | `wontfix`          | 처리하지 않음                 |

스킬이 어떤 역할을 언급할 때(예: "AFK 준비 트리아지 라벨을 적용하라"), 이 표의 해당 라벨 문자열을 사용하세요.

이 저장소는 로컬 마크다운 트래커를 쓰므로, "라벨을 적용한다"는 것은 이슈 파일의 `Status:` 줄에 역할 문자열을 쓰는 것을 의미합니다 (`issue-tracker.md` 참고).

실제로 사용하는 어휘에 맞게 오른쪽 열을 수정하세요.
